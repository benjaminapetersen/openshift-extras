Get ready to rock with OpenShift Enterprise 3.

You're one shell command away from deploying your own Platform as a Service.

Run the installer:

sh <(curl -s https://install.openshift.com/ose)

Looking for Origin?

Origin is the upstream community project for OpenShift Enterprise. There's no better place to develop your new feature and have it battle tested. Check out the Getting Started to see how easy it is to run the platform inside a container.

Take It With You

You can also download the installer and run it from a CD or USB drive. Unpack one of these packages onto your portable media and you are good to go:

  • OpenShift Enterprise Portable: https://install.openshift.com/portable/oo-install-ose.tgz

Need more help? We're here for you.
